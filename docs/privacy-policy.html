Política de Privacidad de VitaAegis

Última actualización: 25 de abril de 2026

1. Introducción

VitaAegis ("la Aplicación") es una aplicación desarrollada por Eulogio Fernández Santiago ("el Desarrollador"). Esta política de privacidad describe cómo la Aplicación gestiona la información del usuario. Nos comprometemos a proteger la privacidad de todos los usuarios, incluidos los menores de edad.

2. Información que recopila la Aplicación

VitaAegis no recopila, transmite ni comparte datos personales con terceros. Toda la información se almacena exclusivamente en el dispositivo del usuario.

La Aplicación procesa localmente los siguientes datos:

  • Notificaciones del dispositivo: La Aplicación lee las notificaciones del sistema para detectar valores o eventos definidos por el usuario en otras aplicaciones. Estos datos se procesan en tiempo real y no se almacenan permanentemente.
  • Preferencias del usuario: Configuraciones como umbrales de alerta, colores, sonidos y personalización del overlay se almacenan localmente en el dispositivo mediante almacenamiento cifrado (EncryptedSharedPreferences).
  • Datos de gamificación: Estadísticas de progreso del usuario (nivel, logros, rachas) se almacenan localmente en el dispositivo de forma cifrada.

3. Datos de menores

VitaAegis puede ser utilizada por menores de edad bajo supervisión de un adulto o tutor legal. La Aplicación:

  • No recopila información personal identificable de ningún usuario, incluidos menores.
  • No contiene publicidad ni enlaces a sitios web externos.
  • No permite compras dentro de la aplicación.
  • No transmite datos fuera del dispositivo.
  • No incluye funciones sociales ni de comunicación entre usuarios.

Cumple con la Ley de Protección de la Privacidad Infantil en Internet (COPPA) y el Reglamento General de Protección de Datos (RGPD) en lo relativo a menores.

4. Permisos requeridos

La Aplicación solicita los siguientes permisos del sistema:

  • Acceso a notificaciones (NotificationListenerService): Para leer valores o eventos mostrados en notificaciones de otras aplicaciones. Ningún contenido de las notificaciones se almacena permanentemente ni se transmite.
  • Superposición sobre otras aplicaciones (SYSTEM_ALERT_WINDOW): Para mostrar el personaje guerrero animado sobre la pantalla del dispositivo.
  • Servicio en primer plano: Para mantener activo el monitoreo de notificaciones mientras la aplicación está en segundo plano.
  • Inicio automático (RECEIVE_BOOT_COMPLETED): Para reiniciar el servicio de monitoreo tras un reinicio del dispositivo, si el usuario lo ha configurado.

5. Almacenamiento y seguridad de datos

Todos los datos se almacenan exclusivamente en el dispositivo del usuario utilizando:

  • EncryptedSharedPreferences con cifrado AES-256 (AES256-SIV para claves, AES256-GCM para valores) para preferencias sensibles de la aplicación.
  • SharedPreferences estándar para preferencias de interfaz no sensibles (colores, tamaño de texto).

La Aplicación tiene deshabilitadas las copias de seguridad automáticas de Android (allowBackup="false") para evitar la extracción no autorizada de datos.

6. Conexión a Internet

VitaAegis no requiere conexión a Internet para funcionar. No se transmiten datos a servidores externos, servicios de analítica ni plataformas publicitarias.

7. Servicios de terceros

La Aplicación no integra servicios de terceros que recopilen datos, como:

  • Google Analytics ni ningún servicio de analítica.
  • Redes publicitarias.
  • Redes sociales.
  • Servicios de crash reporting.

8. Retención y eliminación de datos

Los datos se conservan en el dispositivo mientras la Aplicación esté instalada. Para eliminar todos los datos:

  • Desinstalar la Aplicación, o
  • Borrar los datos de la Aplicación desde Ajustes > Aplicaciones > VitaAegis > Almacenamiento > Borrar datos.

9. Cambios en esta política

Esta política de privacidad puede actualizarse ocasionalmente. Los cambios se publicarán en esta misma página con la fecha de actualización revisada. Se recomienda revisar esta política periódicamente.
